# 画面定義書

## お客様向けWebシステム：申請画面 
### 画面項目
- 申請者名；12文字、テキストボックス
- 住所：100文字、テキストエリア
- 申請内容：200文字、テキストエリア

### 画面レイアウト
~~~text
申請者名；＿＿＿＿＿＿
住所：＿＿＿＿＿＿＿＿＿＿＿＿＿＿
申請内容；＿＿＿＿＿＿＿＿＿＿＿＿
~~~

## 社内担当者向けシステム：申請承認一覧画面
### 画面項目
- 申請番号：最大20桁。形式は「REQ-YYYY-連番6桁」。例：REQ-2026-000001
- 申請日：YYYY/MM/DD
- 承認済：「未」または「済」
- 申請者名：12文字
- 住所：100文字
- 申請内容：200文字

### 画面レイアウト
~~~text
申請番号　          ｜申請日　 ｜承認済｜申請者名｜住所　　　　　　　　｜申請内容
REQ-2026-000001     ｜YYYY/MMDD｜未    ｜山田太郎｜東京都千代田区・・・｜いろはにほへと・・・
・・・
・・・
~~~
