# 画面定義書

## お客様向けWebシステム：申請画面 
### 画面項目
- 申請者；12文字、テキストボックス
- 住所：100文字、テキストエリア
- 申請内容：200文字、テキストエリア

### 画面レイアウト
~~~text
申請者；＿＿＿＿＿＿
住所：＿＿＿＿＿＿＿＿＿＿＿＿＿＿
申請内容；＿＿＿＿＿＿＿＿＿＿＿＿
~~~

## 社内担当者向けシステム：申請承認一覧画面
### 画面項目
- 申請No.：5桁、先頭ゼロ埋め
- 申請日：YYYY/MM/DD
- 承認済：未 or 済
- 申請者：12文字
- 住所：100文字
- 申請内容：200文字

### 画面レイアウト
~~~text
申請者、住所、申請内容、承認済、申請者、住所、申請内容
・・・
・・・
・・・
~~~
